# Multilingual Quick Start

This quick-start sheet is provided in English first and then localized for major YouTube audience markets.

Languages covered here:

- English
- Hindi
- Indonesian
- Portuguese (Brazil)
- Spanish (Mexico)
- Japanese
- German
- Vietnamese
- Filipino
- Turkish

For detailed operation steps, use the full customer and admin guides.

---

## English

1. Log in or activate with your CDK.
2. Create groups before adding servers.
3. Add each server with SSH host, port, username, and password.
4. Add each stream with unique match terms.
5. Use explicit restart commands when possible.
6. Watch the issue panels first, then the group explorer.

## Hindi

1. पहले लॉगिन करें या CDK से सक्रिय करें।
2. सर्वर जोड़ने से पहले समूह बनाएं।
3. हर सर्वर के लिए SSH host, port, username और password भरें।
4. हर stream के लिए unique match terms सेट करें।
5. संभव हो तो explicit restart command दें।
6. सबसे पहले issue panel देखें, फिर group explorer।

## Indonesian

1. Login atau aktifkan akun dengan CDK.
2. Buat grup sebelum menambahkan server.
3. Isi SSH host, port, username, dan password untuk tiap server.
4. Tambahkan stream dengan match terms yang unik.
5. Jika bisa, gunakan restart command yang eksplisit.
6. Cek panel masalah lebih dulu, lalu group explorer.

## Portuguese (Brazil)

1. Faça login ou ative a conta com o CDK.
2. Crie os grupos antes de cadastrar servidores.
3. Informe SSH host, porta, usuário e senha de cada servidor.
4. Cadastre cada stream com match terms exclusivos.
5. Sempre que possível, use um comando de reinício explícito.
6. Veja primeiro os painéis de problema e depois o group explorer.

## Spanish (Mexico)

1. Inicia sesión o activa la cuenta con tu CDK.
2. Crea grupos antes de agregar servidores.
3. Captura host SSH, puerto, usuario y contraseña de cada servidor.
4. Agrega cada stream con match terms únicos.
5. Usa un comando de reinicio explícito cuando sea posible.
6. Revisa primero los paneles de fallas y después el group explorer.

## Japanese

1. ログインするか、CDK でアカウントを有効化します。
2. サーバー追加前にグループを作成します。
3. 各サーバーに SSH host、port、username、password を設定します。
4. 各配信に一意の match terms を設定します。
5. 可能であれば明示的な再起動コマンドを使います。
6. まず異常パネルを見て、その後に group explorer を確認します。

## German

1. Melde dich an oder aktiviere dein Konto mit dem CDK.
2. Lege Gruppen an, bevor du Server hinzufügst.
3. Trage für jeden Server SSH-Host, Port, Benutzername und Passwort ein.
4. Lege für jeden Stream eindeutige Match Terms fest.
5. Verwende wenn möglich einen expliziten Restart-Befehl.
6. Prüfe zuerst die Problem-Panels und danach den Group Explorer.

## Vietnamese

1. Đăng nhập hoặc kích hoạt tài khoản bằng CDK.
2. Tạo nhóm trước khi thêm máy chủ.
3. Điền SSH host, port, username và password cho từng máy chủ.
4. Thêm từng luồng với match terms duy nhất.
5. Nếu có thể, hãy dùng lệnh restart rõ ràng.
6. Xem panel lỗi trước, rồi mới kiểm tra group explorer.

## Filipino

1. Mag-login o i-activate ang account gamit ang CDK.
2. Gumawa muna ng mga group bago magdagdag ng server.
3. Ilagay ang SSH host, port, username, at password ng bawat server.
4. Magdagdag ng stream na may unique na match terms.
5. Kung maaari, gumamit ng explicit restart command.
6. Unahin ang issue panels bago ang group explorer.

## Turkish

1. Giriş yapın veya hesabı CDK ile etkinleştirin.
2. Sunucu eklemeden önce grupları oluşturun.
3. Her sunucu için SSH host, port, kullanıcı adı ve şifre girin.
4. Her yayın için benzersiz match terms tanımlayın.
5. Mümkünse açık bir restart komutu kullanın.
6. Önce sorun panellerini, sonra group explorer görünümünü kontrol edin.
